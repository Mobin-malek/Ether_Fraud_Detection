# Fraud Detection using Machine Learning

This project focuses on detecting fraudulent transactions using machine learning techniques. The goal is to build a classification model capable of distinguishing between legitimate and fraudulent activities based on transaction features.

## Features

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Handling missing values
- Feature scaling and encoding
- Training multiple machine learning models
- Hyperparameter tuning with GridSearchCV
- Model evaluation using:
  - Accuracy
  - Precision
  - Recall
  - F1-score
  - ROC-AUC
- Confusion matrix visualization

## Models Used

- Random Forest Classifier
- XGBoost Classifier



## Dataset

The dataset contains transaction records labeled as either:

- **0** → Legitimate transaction
- **1** → Fraudulent transaction
